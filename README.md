# Django-Web-Project


# Run Locally

```bash
git https://github.com/SlawekKaczmarczyk/Django-Web-Project
```

```bash 
cd mysite/
```

Install dependencies
```bash
#Windows
pip install -r requirements.txt
#Linux
pip3 install -r requirements.txt
```

Start the server

```bash
#Windows
python main.py
#Linux
python3 main.py

# Run 
python manage.py runserver
